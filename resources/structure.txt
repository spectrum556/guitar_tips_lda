Структура файлу: у кожному рядку json об'єкт — група
Структура об'єкта:
{
  "name" — string, назва групи/артиста,
  "views" — integer, кількість переглядів групи (не сума переглядів пісень, а просто к-сть переглядів сторінки групи),
  "songs": [ — array, масив пісень
    {
      "name" — string, назва пісні,
      "views" — integer, к-сть переглядів пісні,
      "text" — string, текст пісні з акордами,
      "comment" — string, коментар автора розбору до пісні
    },
    {...}
  ]
}
